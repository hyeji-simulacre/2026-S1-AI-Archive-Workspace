# 99-trash-bin

파일을 삭제하는 대신 이 폴더로 이동합니다.

## 규칙

- 파일을 완전히 삭제(rm)하지 않는다
- 더 이상 필요 없는 파일은 이 폴더로 이동한다
- AI 에이전트도 파일 삭제 대신 이 폴더로 이동해야 한다
- 정기적으로 검토 후 정말 불필요한 것만 비운다

## 왜 이렇게 하는가

기록학의 원칙: 기록은 함부로 폐기하지 않는다.

실수로 지운 파일을 복구할 수 없으면 돌이킬 수 없다. trash bin을 두면:
- 실수로 지운 파일을 복구할 수 있다
- 언제, 왜 더 이상 필요 없어졌는지 추적할 수 있다
- 나중에 다시 필요해질 수 있다
