# 5주차 AI IDE 설치 기록

5주차 실습에서 AI IDE를 설치하고 첫 실행한 기록.

## 설치 과정

1. Claude Code 설치 완료
2. VSCode에서 폴더 열기
3. 첫 프롬프트: '이 폴더 구조 분석해줘'
4. AI가 폴더 목록을 정확히 읽어냄

## 느낀 점

- 생각보다 설치가 간단했다
- AI가 파일을 직접 읽는다는 게 신기했다
- '구체적으로 말하면 잘 되고, 추상적으로 말하면 안 된다'는 걸 체감

## 아카이브 사유

6주차에서 폴더 구조를 재설계했으므로, 이 기록은 더 이상 현행이 아니다. 과거 기록으로 보존한다.
