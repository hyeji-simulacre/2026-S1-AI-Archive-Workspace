# 에이전트 아이디어 모음

기록관리 실무에서 자동화할 수 있는 문제들을 모아둔다. 7주차 실습에서 이 중 하나를 골라 에이전트로 구현한다.

## 아이디어 목록

### 1. 파일명 날짜 검사기
- 문제: 파일명에 YYYY-MM-DD가 없는 파일이 섞여 있다
- 자동화 적합성: 반복적(O), 규칙 명확(O), 판단 불필요(O)
- 필요한 Tool: Glob, Read

### 2. 태그 통계 분석기
- 문제: 어떤 태그를 얼마나 자주 쓰는지 모른다
- 자동화 적합성: 반복적(O), 규칙 명확(O), 판단 불필요(O)
- 필요한 Tool: Glob, Read, Grep

### 3. 주간 노트 목록 생성기
- 문제: 이번 주에 작성한 노트를 한눈에 보고 싶다
- 자동화 적합성: 반복적(O), 규칙 명확(O), 판단 불필요(O)
- 필요한 Tool: Glob, Read, Write

### 4. 빈 폴더 탐지기
- 문제: 사용하지 않는 빈 폴더가 어디에 있는지 모른다
- 자동화 적합성: 반복적(O), 규칙 명확(O), 판단 불필요(O)
- 필요한 Tool: Glob
