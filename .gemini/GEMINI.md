# AI 워크스페이스 아카이브 규칙

## 폴더 구조
- 00-system: 시스템 설정
- 10-projects: 프로젝트 (하위에 11-프로젝트명/ 형태로 관리)
- 20-created: 내가 만든 것
- 30-collected: 외부 수집 자료
- 40-periodic: 날짜별 기록
- 50-archived: 완료된 과거 자료

## 파일 명명 규칙
- 형식: YYYY-MM-DD-제목.md
- 공백 대신 하이픈(-) 사용

## AI 권한 설정

### 허용
- 모든 폴더의 파일 읽기 (Read)
- 파일 내용 분석 및 요약
- 질문에 대한 답변

### 금지
- 새 파일 생성 (Write) - 금지
- 기존 파일 수정 (Edit) - 금지
- 파일 삭제 (Delete) - 금지
- 터미널 명령 실행 (Bash) - 금지

### 규칙
- 파일을 바꾸고 싶으면 제안만 할 것
- 직접 실행하지 말고 사용자 승인을 기다릴 것
